# Running Coach Web App

My personal running coach web app. I want to use AI to help me become a better runner. I want to use Strava API to get my running data. I want to use Anthropic AI API to be my running coach. I want to use Convex to store my running data. I want to use Clerk to authenticate myself. I want to use NextJS to build my web app. I want to use Tailwind to style my web app.

App should have the following features:
    - Sync data from my strava account.
        - There should be a button to sync data from my strava account for data cost efficiency
    - Data will be analyzed by AI
    - AI will give me a summary of my run and suggest improvements.
        - There should be a button to trigger the AI analysis for data cost efficiency
    - AI will give an overview of my training progress and suggest improvements.
        - There should be a button to trigger the AI analysis for data cost efficiency
    - I can ask the AI to create a training plan for me.
    - There should be a chatbox to chat with the AI. AI should know my strava synced stats so conversation always has context of running.
    - Login via Google
    - Mobile first design
    - Desktop dashboard look
